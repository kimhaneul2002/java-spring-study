# 🚀 Java & Spring Boot 학습 기록 저장소

Java 기초 복습부터 시작하여 Spring Boot 입문, JPA와 MySQL을 활용한 데이터베이스 연동까지의 학습 과정을 기록하는 저장소입니다.

## 📅 학습 로드맵 및 진행 상황
- [x] 1. Java 핵심 개념 복습
- [x] 2. Spring Boot 입문 (기본 MVC 구조 및 API 구조 이해)
- [x] 3. DB 연동 실습 (JPA + MySQL 환경 설정 및 CRUD)
---

## 💻 단계별 학습 내용

### 1. Java 복습
- **핵심 학습:** 객체지향 개념 복습 및 컬렉션 프레임워크 활용.

### 2. Spring Boot 입문
- **핵심 학습:** `@RestController` 기반의 REST API 설계 기초와 레이어드 아키텍처 흐름 이해.

### 3. DB 연동 (JPA + MySQL)
- **핵심 학습:** `application.properties` 설정을 통한 MySQL 연결 및 JPA를 활용한 기본 CRUD 메서드 활용.

---

## 🛠️ 개발 환경 (Tech Stack)
- **Language:** Java
- **Framework:** Spring Boot
- **ORM:** Spring Data JPA
- **Database:** MySQL
